# Link-Manager

| HTTP method |      API endpoint      | Description              |
|-------------|------------------------|--------------------------|
| GET         | /links                 | Get a list of all links. |
| GET         | /links/<link_id>       | Get a single link.       |
| POST        | /links                 | Create a new links       |
| PUT         | /links/<link_id>       | Update a link            |
| DELETE      | /links/<link_id>       | Delete a link            |
